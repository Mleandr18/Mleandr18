### Sobre mim

Sou analista de dados em início de carreira, no momento em que a curiosidade ainda é maior que a experiência — e isso, para mim, é uma vantagem. Estou construindo uma base sólida em análise, engenharia de dados e machine learning, sempre com a mesma pergunta na cabeça: **"o que esses números estão tentando me dizer?"**

Trabalho com Python e SQL no dia a dia, estou expandindo para o universo de engenharia de dados (pipelines, ETL, cloud) e dando os primeiros passos sérios em machine learning. Também mantenho contato com o front-end (HTML, CSS, JS) porque acredito que dado bem analisado merece ser bem apresentado.

Aqui no GitHub você vai encontrar projetos reais, em progresso constante — sem enrolação, sem prints de Excel disfarçados de "ciência de dados". Cada repositório tem documentação, código comentado e uma tentativa honesta de fazer algo útil com dados públicos ou simulados.
